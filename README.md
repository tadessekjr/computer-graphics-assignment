# Maze Generator and Solver

This project  tries to generate and solve a maze using Depth First Search (DFS) and stack-based backtracking.

The maze is created dynamically using a randomized DFS algorithm where a virtual “mouse” removes walls between cells to create a valid path structure.
The solver then automatically traverses the maze while visually showing the current path, dead ends and a little bit of backtracking

## Features

- Random maze generation
- Dynamic maze animation
- Stack-based DFS generation
- Automatic maze solving
- Backtracking visualization
- Dead-end detection
- Random cycle generation (bonus feature)
